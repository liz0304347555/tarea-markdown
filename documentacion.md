# Introducción

Visual Studio Code es uno de los editores de código más utilizados en el mundo debido a su rapidez, facilidad de uso y compatibilidad con múltiples lenguajes de programación.

---

# Requisitos

* Windows 10 o superior.
* Conexión a Internet.
* 500 MB de espacio libre.
* Permisos para instalar programas.

---

# Descarga

Para descargar Visual Studio Code visita el sitio oficial:

https://code.visualstudio.com

---

# Instalación

1. Descargar el instalador desde la página oficial.
2. Ejecutar el archivo descargado.
3. Aceptar los términos de uso.
4. Seleccionar la ubicación de instalación.
5. Completar el proceso de instalación.
6. Abrir Visual Studio Code.

---

# Primer programa

Ejemplo de un programa sencillo en Python:
