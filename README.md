# Instalación

1. Acceder al sitio oficial de Visual Studio Code.
2. Descargar el instalador correspondiente al sistema operativo.
3. Ejecutar el archivo descargado.
4. Aceptar los términos de licencia.
5. Seleccionar la ubicación de instalación.
6. Completar la instalación.
7. Abrir Visual Studio Code.

# Parte 4 — Reflexión

## ¿Por qué Markdown es tan utilizado en programación y tecnología?

Markdown es muy utilizado porque permite crear documentación de forma rápida, sencilla y organizada. Además, es compatible con plataformas como GitHub y otras herramientas empleadas por desarrolladores. Su sintaxis es simple y fácil de aprender.

## ¿En qué situaciones preferirías Markdown sobre herramientas visuales como Notion?

Preferiría Markdown cuando necesito crear documentación técnica, manuales o archivos ligeros que puedan compartirse fácilmente entre distintas plataformas y sin depender de Internet.

## ¿Qué ventajas ofrecen las plataformas colaborativas frente a Markdown tradicional?

Permiten que varias personas trabajen simultáneamente sobre un mismo documento, ofrecen comentarios, historial de cambios y almacenamiento en la nube.

## ¿Crees que Markdown seguirá siendo relevante en el futuro? ¿Por qué?

Sí. Markdown seguirá siendo relevante debido a su simplicidad, compatibilidad y amplia adopción en el ámbito de la programación y la documentación técnica.
